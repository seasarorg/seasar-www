S2Ebi Easy(容易な) Business(商売) Imperative(規則) なルール処理ライブラリ (準備中)

S2Laszlo FlashプレーヤーからLaszloを用いてS2Containerのコンポーネントを呼び出すライブラリ
S2Mai アスペクトを使用してメール送信するライブラリ
S2Portlet S2Container を JSR 168 準拠のポートレット上で利用するためのライブラリ
S2Sao アスペクトを使用してJavaScriptのロジックを実行するためのライブラリ (準備中)
S2SqlMap iBATIS SQL Maps対応ライブラリ

S2XWork 汎用コマンドパターンフレームワークXWorkをS2Containerと連携させるためのライブラリ
Seasar-genapp S2Containerを利用する開発プロジェクトの雛形を生成するMavenプラグイン (準備中)
Tuigwaa HTML タグを知らない人が、専門知識や特別な環境なしに ブラウザだけでデータベースと連動する Web　アプリケーションを作るツール
Uuji 規約重視によって必要最小限のコードで永続化処理を行う (準備中)
