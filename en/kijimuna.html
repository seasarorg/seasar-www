Kijimuna enables programmers to easily verify dicon files enabling programmers to find errors before dicon files are actually used.

Setup

Eclipse3 or later is required to use Kijimuna. Extract Kijimuna archive files and move them under Eclipse plugins directory.

How to Use

  • Open Problems View
  • In Filter setting, check S2 Problem Marker
  • In Project Properties, check Seasar2 Project
  • Rebuild project. Project->Clean...->Clean selected project.
